HTML문서 중 특정 부분을 선택하여 해당 부분의 속성을 변경하는 목적
tag선택자 : 특정태그를 선택하여 CSS속성을 적용할 수 있다.

* 선택자 : 전체라는 의미의 * 선택자를 사용하면 문서 전체를 선택하는 의미가 있다 

id(#)와 class(.) : 태그의 id와 clas를 이용하여 CSS에 속성을 설정할 수 있다

속성 선택자 : tag의 속성을 선택하여 CSS의 속성을 설정할 수 있다
(1) 선택자[속성] : 지정한 속성을 가지고 있는 요소를 찾아 스타일을 적용
(2) 선택자[속성=값] : 속성 값이 특정값과 같은 경우를 찾아 스타일을 적용
(3) 선택자[속성^=값] : 속성 값이 지정한 문자로 시작하는 속성값에 대해서만 스타일을 적용
(4) 선택자[속성$=값] : 속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일을 적용

동위(형제) 선택자 (+ 하나만, ~부터 다선택)  : 태그의 동등한 위치를 판단하여 CSS속성을 설정
ex. h3+div{} : h3 바로 다음에 오는 div(형제)
ex. h3~div{} : h3 다음에 오는 div(형제)전부

반응 선택자 : 마우스를 올려 놓았을 때(hover상태)의 반응에 따른 속성을 설정
ex. nav ul li:hover{color:pink;} 마우스 커서를 올리면 핑크

상태 선택자 : 상태에 따라 CSS속성이 변화하는 설정을 할 수 있다.  
ex. input태그의 경우 focus가 맞혀졌을 때

구조 선택자 : 구조에 따라 CSS 속성이 변화하는 설정을 할 수 있다
ex. nth-child(2n+1){} : 홀수번째 /nth-child(2n):짝수번째
